# Node-issues

Get NVM:
`curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.1/install.sh | bash` </br>
Install Node Version
`nvm install v16.15.0`



NVM version change
`nvm use v16.15.0` </br>
NVM default version change
`nvm alias default v16.15.0`
